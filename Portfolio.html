Quote Generator

Based on one of my favorite books, The Dog Stars by Peter Heller, this site will generate random quotes from the book when the button is clicked. If desired, you can tweet the quote using the Tweet button to the right. Built using Bootstrap and Jquery.

Wikipedia Search

This provides a simple way to search for Wikipedia articles- enter your search term and a list of relevant articles is shown below. You can also click 'Take Me To A Random Page' to view a random article. Built using Angular JS.

Twitch Viewer

Using the Twitch API, you can view the current streamers on Twitch TV and what they're streaming. Click on the streamer to go to their Twitch page. Built using Angular JS.
